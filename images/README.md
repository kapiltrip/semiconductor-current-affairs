# Images Index

This folder stores dated screenshots for daily semiconductor current-affairs notes.

Use one folder per date:

```text
images/YYYY-MM-DD/
```

Each date folder must contain a `links.md` manifest with:

- screenshot filename
- source name
- source URL
- source date
- topic
- screenshot status

Screenshot and editorial coverage policy:

- Capture only headline, date, source name, and short visible snippet.
- Embed each relevant image/screenshot in the matching daily note before its explanation.
- Use screenshots to preserve source context and make the note easier to study.
- Cover related editorials fully in the daily note using original study writing: thesis, arguments, evidence, counterpoints, semiconductor/VLSI relevance, India angle, and revision questions.
- Keep source links in `links.md` so the daily note stays verifiable.
