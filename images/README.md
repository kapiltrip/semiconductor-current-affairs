# Images Index

This folder stores dated screenshots for daily semiconductor current-affairs notes.

Use one folder per date:

```text
images/YYYY-MM-DD/
```

Each date folder must contain a `links.md` manifest with:

- screenshot filename
- source name
- source URL
- source date
- topic
- screenshot status

Screenshot policy:

- Capture only headline, date, source name, and short visible snippet.
- Do not capture full article bodies.
- Do not capture full paywalled pages.
- Keep source links in `links.md` so the daily note stays verifiable.
